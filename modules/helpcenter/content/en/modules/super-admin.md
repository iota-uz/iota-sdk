# Super Admin

Super Admin is used to oversee tenants and platform-level activity. It is intended for teams that operate IOTA SDK as a multi-tenant platform.

## Who uses it

- Platform operators
- SaaS administrators
- Support leads
- Business owners responsible for tenant health

## What it manages

- Tenant overview
- Tenant users
- Tenant analytics
- Platform dashboard metrics
- High-level usage and health information

## Why it matters

When several organizations use the same platform, operators need visibility across tenants without entering every workspace manually. Super Admin helps teams understand tenant growth, activity, and support needs.

## Common tasks

- Review the tenant list.
- Inspect tenant-level usage.
- Check users within a tenant.
- Use dashboard metrics to monitor platform health.
- Support a tenant when an access or configuration issue occurs.

## Best practices

- Restrict Super Admin access to platform-level roles.
- Use tenant context carefully.
- Keep support actions auditable.
- Review tenant activity trends regularly.
