# API Module

The API module provides GraphQL and REST API endpoints for platform integration.

## Features

This module includes essential features for api management.

## Usage

The API module is registered automatically when you include it in your application setup. Keep service wiring and transport registration separate inside your module:

```go
func (m *Module) RegisterWiring(app application.Application) error {
	return nil
}

func (m *Module) RegisterTransports(app application.Application) error {
	app.RegisterControllers(
		controllers.NewAPIController(app),
	)

	app.RegisterGraphSchema(application.GraphSchema{
		Value:    schema,
		BasePath: "/api",
	})

	return nil
}
```
