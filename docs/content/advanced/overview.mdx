# Advanced Module

The advanced module provides functionality for IOTA SDK platform.

## Features

This module includes essential features for advanced management.

## Usage

Load a module by passing it to `modules.Load`; the module's `Register(app)` is called internally:

```go
modules.Load(app, yourModule)
// or with built-ins: modules.Load(app, modules.BuiltInModules..., yourModule)
```
