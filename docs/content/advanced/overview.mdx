# Advanced Module

The advanced module provides functionality for IOTA SDK platform.

## Features

This module includes essential features for advanced management.

## Usage

Compose modules explicitly so wiring, transports, and runtime startup stay separate:

```go
allModules := append([]application.Module{}, modules.BuiltInModules...)
allModules = append(allModules, yourModule)

if err := application.Wire(app, allModules...); err != nil {
	return err
}
if err := application.RegisterTransports(app, allModules...); err != nil {
	return err
}
if err := app.StartRuntime(
	context.Background(),
	application.RuntimeTagAPI,
	application.RuntimeTagWorker,
); err != nil {
	return err
}
```
