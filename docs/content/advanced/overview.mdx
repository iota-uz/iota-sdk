# Advanced Module

The advanced module provides functionality for IOTA SDK platform.

## Features

This module includes essential features for advanced management.

## Usage

Compose modules with a profile so wiring, transports, and runtime startup happen explicitly. For most long-running servers, use `application.ApplyProfile`:

```go
allModules := append([]application.Module{}, modules.BuiltInModules...)
allModules = append(allModules, yourModule)

if err := application.ApplyProfile(
	context.Background(),
	app,
	application.CompositionProfileServer,
	allModules...,
); err != nil {
	return err
}
```
