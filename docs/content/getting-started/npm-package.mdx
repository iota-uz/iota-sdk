# NPM Package (`@iota-uz/iota-sdk`)

`@iota-uz/iota-sdk` is the unified public NPM package that ships:
- React utilities for applets (`@iota-uz/iota-sdk`)
- BIChat UI (`@iota-uz/iota-sdk/bichat`)
- Tailwind v4 inputs + config helper (`@iota-uz/iota-sdk/tailwind/*`)
- Bundled assets (fonts) under `@iota-uz/iota-sdk/assets`

## Install

```bash
pnpm add @iota-uz/iota-sdk
pnpm add -D tailwindcss@4.1.18 @tailwindcss/cli@4.1.18
```

## Tailwind v4 (consumer compiles)

### 1) Create Tailwind config

```js
// tailwind.config.cjs
const createIotaTailwindConfig = require('@iota-uz/iota-sdk/tailwind/create-config')

module.exports = createIotaTailwindConfig({
  content: ['./src/**/*.{ts,tsx,js,jsx,html}'],
  extend: {},
})
```

### 2) Create a Tailwind input CSS file

Tailwind v4 loads config via `@config`, so your input CSS should reference your config and then import the SDK Tailwind input:

```css
/* src/tailwind.css */
@config "../tailwind.config.cjs";
@import "@iota-uz/iota-sdk/tailwind/main.css";

/* Add your own layers/custom CSS below if needed */
```

### 3) Build CSS

```bash
pnpm exec tailwindcss --input ./src/tailwind.css --output ./public/assets/main.css --minify
```

## BIChat UI

```ts
import { BiChatApp } from '@iota-uz/iota-sdk/bichat'
import '@iota-uz/iota-sdk/bichat/styles.css'
```

## Assets (fonts)

If you use SDK fonts, serve `node_modules/@iota-uz/iota-sdk/assets` at `/assets` so URLs like `/assets/fonts/...` resolve.
